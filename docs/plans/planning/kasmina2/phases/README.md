# Kasmina2 Phases

This folder is the phase-by-phase execution workspace for Kasmina2 (Tracks A + C).

- Phase details and rationale live in `docs/plans/planning/kasmina2/kasmina_tamiyo_submodule_intervention_roadmap.md`.
- Each phase folder decomposes that roadmap into **dev-sized, cross-domain checklists** (Leyline → Kasmina → Tamiyo → Simic → Tolaria/runtime → Karn/Nissa).
- When a phase is ready to implement, promote it to a single plan under `docs/plans/ready/` (no backcompat means the plan must list *all* call sites that change).
