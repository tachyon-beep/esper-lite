## Detailed Description

### What's Wrong

```python
# Line 192
if self.counterfactual_helper is not None:
    self.counterfactual_helper._last_matrix = None
```

Accessing `_last_matrix` (private by Python convention with leading underscore) violates encapsulation. The `CounterfactualHelper` class should expose a public `reset()` method.

### Impact

- **Fragile coupling**: Changes to CounterfactualHelper internals break this code
- **Hidden contract**: Callers must know about private implementation details
- **Testing difficulty**: Can't mock/test CounterfactualHelper independently

---

## Recommended Fix

Add a public reset method to CounterfactualHelper:

```python
# In CounterfactualHelper class:
def reset(self) -> None:
    """Reset cached state for new episode."""
    self._last_matrix = None

# In ParallelEnvState:
if self.counterfactual_helper is not None:
    self.counterfactual_helper.reset()
```
