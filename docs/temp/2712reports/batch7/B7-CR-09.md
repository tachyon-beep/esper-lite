## Detailed Description

### What's Wrong

```python
def should_collect(self, category: str) -> bool:
    # ... checks known categories ...
    return False  # Unknown categories silently return False
```

This means typos are not caught:
```python
config.should_collect("debug")   # Returns False or True (valid)
config.should_collect("degub")   # Returns False (typo, silently ignored)
config.should_collect("gradients")  # Returns False (is this valid?)
```

Users have no way to know if a category name is invalid vs just disabled.

---

## Recommended Fix

**Option 1 - Raise for unknown:**
```python
VALID_CATEGORIES = {"debug", "per_layer", "analytics", ...}

def should_collect(self, category: str) -> bool:
    if category not in VALID_CATEGORIES:
        raise ValueError(f"Unknown telemetry category: {category!r}")
    # ...
```

**Option 2 - Log warning:**
```python
def should_collect(self, category: str) -> bool:
    if category not in VALID_CATEGORIES:
        _logger.warning("Unknown telemetry category: %s", category)
        return False
    # ...
```
