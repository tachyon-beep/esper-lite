## Detailed Description

### What's Wrong

```python
@property
def normalized_ratio(self) -> float:
    """Ratio of module grad to host grad, normalized by param count."""
    if self.module_grad_norm <= 0 or self.host_grad_norm <= 0:
        return 0.0  # No gradients case
    if self.module_param_count == 0 or self.host_param_count == 0:
        return 0.0  # Division by zero protection
    # ...
```

The return value 0.0 is ambiguous:
1. **Zero gradients** → 0.0
2. **Equal gradients** → ratio ≈ 1.0, but normalized could be 0.0 if param counts equal
3. **No params** → 0.0

This makes it impossible to distinguish "no data" from "equal gradients" from "zero gradients".

---

## Recommended Fix

Use sentinel values or Optional return:

```python
@property
def normalized_ratio(self) -> float | None:
    """Ratio of module grad to host grad, normalized by param count.

    Returns None if gradients are unavailable (zero or missing).
    """
    if self.module_grad_norm <= 0 or self.host_grad_norm <= 0:
        return None  # No gradients
    if self.module_param_count == 0 or self.host_param_count == 0:
        return None  # No params
    # ... compute actual ratio ...
```
