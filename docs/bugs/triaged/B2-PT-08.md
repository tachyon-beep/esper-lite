## Detailed Description

### What's Wrong

The error message at L305 references "num_segments" which is a local variable. Users seeing this error may be confused about where this value comes from.

### Code Evidence

```python
# /home/john/esper-lite/src/esper/kasmina/host.py:305

if n_layer % num_segments != 0:
    raise ValueError(f"n_layer ({n_layer}) must be divisible by num_segments ({num_segments})")
```

### Why This Matters

Minor clarity issue - users would understand better with:
- "number of injection segments" instead of "num_segments"
- Or reference to how num_segments is derived from `injection_points`

---

## Recommended Fix

```python
if n_layer % num_segments != 0:
    raise ValueError(
        f"n_layer ({n_layer}) must be divisible by the number of injection segments "
        f"({num_segments}, derived from {len(injection_points)} injection points)"
    )
```
