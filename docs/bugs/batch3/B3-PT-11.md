## Detailed Description

### What's Wrong

The `NoopSeed.__init__` accepts a `channels` parameter for API consistency with other seeds, but doesn't use it since noop has no parameters.

### Code Evidence

```python
# /home/john/esper-lite/src/esper/kasmina/blueprints/cnn.py:24

def __init__(self, channels: int):
    super().__init__()
    # No parameters - pure pass-through
```

## Resolution
