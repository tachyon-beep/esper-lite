## Detailed Description

### What's Wrong

The `NoopSeed.__init__` accepts a `channels` parameter for API consistency but doesn't use it. While harmless, this is misleading.

### Code Evidence

```python
# /home/john/esper-lite/src/esper/kasmina/blueprints/cnn.py:23-29

class NoopSeed(nn.Module):
    def __init__(self, channels: int):  # Accepted but unused
        super().__init__()
        # No parameters - pure pass-through

    def forward(self, x: torch.Tensor) -> torch.Tensor:
        return x
```

---

## Recommended Fix

Either remove the parameter or document why it's kept:

```python
class NoopSeed(nn.Module):
    def __init__(self, channels: int):
        """Pure pass-through seed. channels is accepted for API consistency."""
        super().__init__()
        del channels  # Explicit: unused, for registry API consistency
```
